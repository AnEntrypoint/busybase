Quick Start

From zero to running in under 60 seconds.

No config files. No database setup. No Docker.

# 1. Start the server
npx busybase serve

# 2. Install the SDK
bun add busybase

# 3. Connect and query
import BB from 'busybase';
const db = BB('http://localhost:54321', 'local');

const { data } = await db.from('todos').insert({ title: 'Buy milk', done: false });
const { data: todos } = await db.from('todos').select('*').eq('done', false);
await db.from('todos').update({ done: true }).eq('title', 'Buy milk');

# 4. Open Studio UI
open http://localhost:54321/studio
